# Week 8 — Exercises

Three exercises. Each is solo writing or solo scripting work that rehearses one of the day-2 skills. Together they build the muscle for the Saturday-Sunday dry-run continuation.

| Exercise | Skill rehearsed | Time | When |
|---------|------------------|------|------|
| [exercise-01-pivot-decision-drill.md](./exercise-01-pivot-decision-drill.md) | The 15-minute pivot decision protocol; applied to three fake scenarios. | ~1h | Tuesday |
| [exercise-02-integration-friction-drill.md](./exercise-02-integration-friction-drill.md) | The merge protocol; resolving conflicts on a fake repo state with starter code. | ~1h | Friday |
| [exercise-03-retro-from-fake-log.md](./exercise-03-retro-from-fake-log.md) | The hour-24 retrospective; writing a retro and follow-up issue from a fake `DAY-2-LOG.md`. | ~30 min | Friday |

Solutions for all three are in [SOLUTIONS.md](./SOLUTIONS.md). Do the exercises first; cover the solutions; check after.

## Submission

Each exercise's deliverable is a Markdown file committed to your `week-08/` working folder in the dry-run team's repo (the same repo as `DAY-1-LOG.md`). The Comms Lead can collect all three under a single `WEEK-08-EXERCISES.md` if that is easier.

## Voice

The C4 voice on these exercises is sober, decision-focused, and time-boxed. The exercises are not creative writing; they are *judgment under constraint*. Aim for clarity over polish.

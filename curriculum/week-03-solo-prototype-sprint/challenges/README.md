# Week 3 — Challenges

The mini-project is the solo build. **The challenge is the same build, paired.** It is optional, but if you can find another C4 learner with a free Saturday, it is the single best way to see your scoping habits from the outside — which is exactly what Week 4 (Agile ceremonies, team-mode) is about to require.

## Index

1. **[Challenge 1 — Pair Prototype](challenge-01-pair-prototype.md)** — run the same six-hour solo build paired with another C4 learner, on parallel prompts, with a 60-second sync at every hourly stand-up. (~6 hours of build + 30 minutes of pairing logistics)

The challenge is optional. If you skip it, the solo build still passes Week 3. If you do the pair version, you arrive at Week 4 with a head start on the team-mode rhythm.
